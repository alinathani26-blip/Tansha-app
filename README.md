## WHAT'S IN THIS FOLDER

```
tansha-app/
├── public/
│   ├── index.html       ← App shell
│   └── manifest.json    ← PWA config (save to home screen)
├── src/
│   ├── index.js         ← Entry point
│   └── App.jsx          ← Full app (all 10 modules)
├── package.json         ← Dependencies
└── vercel.json          ← Deployment config
```

---

## STEP 1 — Create GitHub Account (free)
1. Go to **github.com**
2. Sign up with your email
3. Create a new repository called `tansha-app`
4. Upload all files from this folder

---

## STEP 2 — Deploy on Vercel (free)
1. Go to **vercel.com**
2. Sign up with GitHub (same account)
3. Click **"Add New Project"**
4. Select your `tansha-app` repository
5. Click **Deploy**
6. In ~2 minutes you get a link like: `tansha-app.vercel.app`

✅ **That's it. App is live.**

---

## STEP 3 — Share with Team
Send the link to all 17 team members.
They open it in browser → tap **"Add to Home Screen"** → works like an app.
